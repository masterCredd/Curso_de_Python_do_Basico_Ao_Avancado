# **Configurar o ambiente do python na ultima versão**

___

## Configurar o ambiente do python na ultima versão

* Utilizando o **terminal** digite o seguinte
  📝 `comando:`

    ```bash
    docker pull python
    ```

* Confirmar a versão instalada

     📝`comando:`

    ```bash
   python --version
    ```

!!!Success  "✔️ Resposta:"

    ```bash
      Python 3.11.0 # ou superior
    ```

* Instale o pacote a baixo:

  ```bash
    pip install -r requirements.txt
  ```
